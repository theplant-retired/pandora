## How to use

- Setup [Haskell Plaform] (http://www.haskell.org/platform/). 

- Makesure [cabal](http://www.haskell.org/cabal/) is installed 

- Setup pandora

```bash
git clone git@github.com:theplant/pandora.git
cd pandora
cabal install
cabal configure
cabal build
cabal install
```

- Run services

```
pandora
```
